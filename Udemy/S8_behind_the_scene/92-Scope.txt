What is scope?
The place where a variable is declared.
There are global scope, function scope and block scope.

What is scoping?
it controlls how our program's variables are organized and accessed.

What is Lexical scoping?
Scoping is controlled by placement of functions and blocks in the code.

What is scope of a variable?
The Region of the code where a variable can be accessed.
